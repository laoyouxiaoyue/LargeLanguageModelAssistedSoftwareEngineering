Watermark Application - Windows Release
=====================================

使用方法：
1. 双击 run.bat 启动应用
2. 或者直接双击 watermark-app.exe 启动应用

功能特性：
- 支持单张图片和文件夹批量处理
- 实时预览水印效果
- 可调整字体大小、颜色、位置
- 支持模板保存和加载
- 支持多种图片格式（JPEG、PNG）

系统要求：
- Windows 10 或更高版本
- 无需安装额外依赖

注意事项：
- 首次运行可能需要几秒钟启动时间
- 建议在处理大量图片时耐心等待
- 支持拖拽图片到应用窗口

技术支持：
如有问题，请检查：
1. 确保图片格式受支持
2. 确保有足够的磁盘空间
3. 确保有文件写入权限
